The notice shortcode shows 4 types of disclaimers to help you structure your page.

## Note

```
{{%/* notice note */%}}
A notice disclaimer
{{%/* /notice */%}}
```

Renders as:

{{% notice note %}}
A notice disclaimer
{{% /notice %}}

## Info

```
{{%/* notice info */%}}
An information disclaimer
{{%/* /notice */%}}
```

Renders as:

{{% notice info %}}
An information disclaimer
{{% /notice %}}

## Tip

```
{{%/* notice tip */%}}
A tip disclaimer
{{%/* /notice */%}}
```

Renders as:

{{% notice tip %}}
A tip disclaimer
{{% /notice %}}

## Warning

```
{{%/* notice warning */%}}
A warning disclaimer
{{%/* /notice */%}}
```

Renders as:

{{% notice warning %}}
A warning disclaimer
{{% /notice %}}
